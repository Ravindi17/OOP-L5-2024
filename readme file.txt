# Ticketing System

A simple ticketing system where vendors release tickets to a pool, and customers retrieve tickets from the pool.

## Table of Contents
- [Introduction]
- [Setup]
- [Usage]

## Introduction

This project simulates a ticket distribution system where vendors release tickets to a pool, and customers can retrieve tickets based on availability. The system uses Java and employs a `ConcurrentLinkedQueue` to manage ticket storage.

## Setup

### Prerequisites
- Java with maven for the CLI
- Spring for the backend API
- ReactJS for the frontend UI
